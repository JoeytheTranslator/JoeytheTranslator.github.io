【傻瓜教程】如何在Windows11安装安卓应用

2023-11-09

能解决的问题：某些平台或者App没有网页版和网页版做的太烂

  • 本人有听播客的需求。
  • Windows平台貌似没有特别现代的免费播客软件。Grover Podcast还可以，但是用起来有时候会卡顿，或者因故无法播放。
  • 在移动端，更早之前是用的是Castro，但是在大陆被禁了，于是逐步尝试其他软件，并最终转向小宇宙。但是，小宇宙没有网页版，平常在笔记本电脑上没法同步。
  • 偶然发现Windows可以安装安卓应用。便萌生了把小宇宙安卓版安装到Windows的想法。

以下为实现步骤：

  1. 打开微软商店（Microsoft Store），搜索Amazon Appstore，点击安装，无脑点击下一步，等待安装，然后重启电脑。重启后，【适用于 Android 的 Windows 子系统】已经安装完成；
  2. 在子系统设置页面开启开发者人员模式；
  3. 安装 WSA PacMan;
  4. 下载 APK 文件。比如，在 Apkpure 搜索小宇宙并下载 APK 文件；
  5. 双击 APK 文件即可安装软件。
